# Light Restaurant Website Project

A premium restaurant website featuring Home, Menu, About, and Contact pages with elegant, fluid layouts, custom navigation, and asset serving.

## Features

- **Home Page**: A beautiful fast-food showcase with hero sliders and featured grids.
- **Menu Page**: A detailed menu catalog highlighting special dishes.
- **About Page**: Information about restaurant services and standards.
- **Contact Page**: Reach us via reservation details and location maps.
- **Local Dev Server**: Served using Express to efficiently route all local pages and shared resources.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the website:
   Navigate to `http://localhost:3005` in your browser.
